📅 Day 1: OOP Concepts, Java Features & Selenium Overview

Programming Paradigms

  • Structured: C, Python – logical flow.
  • Object-Based: VB, VBScript – classes/objects but no inheritance.
  • Object-Oriented (OOP): Java, C++, C# – supports Class, Object, Polymorphism, Inheritance, Abstraction, Encapsulation.

Java Features

  • Platform Independent (JVM + Bytecode).
  • Case-sensitive, Robust, Multithreaded, Secure.
🧠 JDK, JRE, JVM: JDK (development tools) includes JRE. JRE (libraries + JVM) runs apps. JVM executes bytecode → platform independence.

Java Naming & Comments

  • Class names: start with uppercase, no special chars (except _).
  • Comments: // single-line , /* multi-line */.
  • Eclipse shortcut: Syso + Ctrl+Space → System.out.println();

📅 Day 3: Operators in Java

  • Arithmetic: + - * / %

📅 Day 9: String Reverse, Comparison, Immutability & Assignments

Reverse a string

String s = "welcome";
StringBuilder sb = new StringBuilder(s);
String reversed = sb.reverse().toString(); // "emoclew"

== vs equals()

  • == compares object references (memory).
  • equals() compares content – recommended for string comparison.

Immutability / Mutability

  • String: Immutable – any change creates new object.
  • StringBuffer: Mutable, thread-safe.
  • StringBuilder: Mutable, faster (not thread-safe).
❓ Key doubt: StringBuilder s = new StringBuilder("welcome"); String b = s.reverse();
Answer: StringBuilder is mutable, reverse() modifies the original object, so both s and b refer to "emoclew".

🧪 Selenium – Web Automation Suite

Selenium is an open source tool for automating web browsers across platforms (Windows, Mac, Linux) and browsers (Chrome, Firefox, Safari, Edge).

Components of Selenium Suite

  • Selenium IDE: Record/playback add-on (Chrome/Firefox).
  • Selenium RC (Remote Control): Selenium 1 (deprecated).
  • WebDriver: Direct browser communication (core, modern).
  • Selenium Grid: Parallel execution across multiple machines/browsers.

Advantages

  • Free & open source, multi-language support (Java, Python, C#, Ruby, JS).
  • Integrates with TestNG, JUnit, Jenkins.
  • Cross-browser & cross-platform.

Limitations

  • Only web apps (no desktop apps).
  • No built-in reporting, no image testing (needs Sikuli).

🛠 JDK 22 & Eclipse Installation (Windows & Mac)

Windows

  • Download jdk-22_windows-x64_bin.exe from Oracle.
  • Run installer, default location: C:\Program Files\Java\jdk-22.
  • Set Environment Variables: add JAVA_HOME and update Path with %JAVA_HOME%\bin.
  • Verify: java -version in command prompt.

Mac (Silicon M1/M2)

  • Download jdk-22_macos-aarch64_bin.dmg → install .pkg.
  • Set path in ~/.zshrc:
    export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-22.jdk/Contents/Home
  • Run source ~/.zshrc then echo $JAVA_HOME.

Eclipse IDE

  • Download "Eclipse IDE for Java Developers" from eclipse.org.
  • Launch and select JDK 22 as the VM.
  • Create project: File → New → Java Project → Package → Class.
